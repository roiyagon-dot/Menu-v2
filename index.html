⚙️ Configurar sincronización

Pasos (5-10 min, solo una vez):
1. Ve a console.cloud.google.com → crea un proyecto
2. APIs & Services → Enable APIs → busca "Google Sheets API" → activar
3. Credentials → Create Credentials → API Key → copia la clave
4. Crea un Google Sheet nuevo → compartir como "cualquiera con el enlace puede editar"
5. Copia el ID del Sheet (la parte larga en la URL entre /d/ y /edit)
6. Pega ambos aquí abajo:

Tus credenciales se guardan solo en este dispositivo (localStorage). Tendrás que configurarlo también en el móvil la primera vez.
